## How to Use Downloaded Images

### Step 1: Prepare the Image
1. Download high-resolution image (at least 1920x1080)
2. Optimize for web using:
   - ImageOptim (Mac) / PNGCrush (Windows)
   - Or online: TinyPNG, Squoosh
   - Target: Under 500KB for best performance

### Step 2: Place in Public Folder
```bash
# Copy to public folder
cp your-image.jpg client/public/turf-nightlight.jpg
```

### Step 3: Update Hero Component
Replace line 11 in `client/src/components/landing/Hero.jsx`:

```jsx
// Change from:
backgroundImage: `...url(/stadium-hero.jpeg)...`

// To:
backgroundImage: `...url(/turf-nightlight.jpg)...`
```

## Image Requirements

✅ **Ideal Image Characteristics:**
- **Resolution**: 1920x1080 or higher
- **Aspect Ratio**: 16:9 or 21:9
- **Subject**: 
  - Professional football/soccer pitch
  - Illuminated by nightlights or stadium lights
  - Well-maintained turf (fresh, vibrant green)
  - Multiple light angles creating dynamic shadows
- **Quality**: Professional photography
- **Lighting**: Warm or cool nightlight tones (not midday)

❌ **Avoid:**
- Daytime images
- Poor quality/low resolution
- Heavy text overlay
- Faces/people as main focus
